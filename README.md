# Clinic Mobile App 🏥

نظام إدارة العيادات - تطبيق موبايل Flutter

## Screens

1. **Splash Screen** - شاشة البداية
2. **Login Screen** - تسجيل الدخول
3. **Home Screen** - الشاشة الرئيسية
4. **Patients Screen** - قائمة المرضى
5. **Doctors Screen** - الأطباء
6. **Appointments Screen** - المواعيد
7. **Invoices Screen** - الفواتير
8. **Settings Screen** - الإعدادات
9. **Add Patient Screen** - إضافة مريض

## Installation

```bash
flutter pub get
flutter run
```
